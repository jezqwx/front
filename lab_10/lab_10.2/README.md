# Лабораторная работа 10.2  
## Описание
В данной лабораторной работе выполнен деплой React-приложения Todo List, созданного в Lab 10.1.

Также реализованы:
- production build
- конфигурация для деплоя
- настройка CI/CD

## Используемые технологии

- React
- TypeScript
- Vite
- Vercel
- GitHub Actions

## Функциональность
Приложение позволяет:

- Добавлять задачи
- Отмечать задачи как выполненные
- Удалять задачи
- Отслеживать количество задач

## Production build
