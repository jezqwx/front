# Лабораторная работа 10.1  
## Описание

В данной лабораторной работе реализовано React-приложение "Todo List" и написаны модульные тесты с использованием Jest и React Testing Library.

Основная цель — изучить принципы unit-тестирования и проверить корректность работы компонентов.

## Используемые технологии

- React
- TypeScript
- Vite
- Jest
- React Testing Library

## Функциональность приложения
- Отображение списка задач
- Добавление новой задачи (через кнопку и Enter)
- Запрет добавления пустой задачи
- Отметка задачи как выполненной
- Удаление задачи
- Подсчёт общего количества задач и выполненных

## Тестирование
Реализованы тесты для:

- Рендера компонента
- Отображения начального списка задач
- Добавления задач
- Обработки клавиши Enter
- Запрета пустых задач
- Переключения статуса (completed)
- Удаления задач
- Обновления счётчика
