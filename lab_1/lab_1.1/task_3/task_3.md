В лаб 1.1 была выполнена одна и та же задача двумя разными способами: с использованием jQuery и с React. Задача заключалась в том, чтобы по нажатию на кнопку абзац текста получал класс подсветки, а при повторном нажатии этот класс убиралась.
В первой части задания использовался императивный подход с иблиотекой jQuery. В этом случае нужно было вручную работать с DOM:находить кнопку и абзац, вешать обработчик события на кнопку и при нажатие добавлять или убирать CSS класс. Такой подход требует подробно описовать каждый шаг изменения интерфейса. 
Во второй части задания использовался декларативный подход с библиотекой React. Здесь изменения интерфейса происходят через state. В компоненте хранится переменная, которая отвечает за подсветку текста. При нажатии на кнопку состояние изменяется, а React автоматически обновляет интерфейс.
Основное отличие между этими подходами заключается в том что в jQuery разработчик сам управляет изменениями DOM, а в React он описывает, как интерфейс должен выглядеть в зависимости от состояния
